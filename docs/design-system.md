# Tandela Design System

## Temel tokenlar

- Arka plan: `#F8F9F7`
- Kart: `#FFFFFF`
- Marka: `#4A7C63`
- CTA: `#2D5A45`
- Border: `#DDE3E0`

## Section ve Chip desenleri

- Section zemini: büyük bölüm arka planlarında soft sage `#E8EDE8` kullanılır.
- Numaralı bölüm başlığı: bölüm numarası `01` formatında, açık gri `#9CA89F`,
  `32px`; yanında başlık koyu metinle gösterilir.
- Kondisyon chip'i: beyaz kart, `rounded-lg`, `shadow-sm`; solda 3px dikey
  kondisyon rengi çizgisi bulunur. İçerik: kondisyon adı `font-weight: 500`,
  diş no, varsa yüzey harfleri `tabular-nums`, ve yalnız onay öncesi
  düzenlenebilir modda silme `X`.
